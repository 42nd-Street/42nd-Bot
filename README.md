# 42nd Street Bot

In development

Current stack:

- Node.js
- discord.js
- Typescript
- Docker (w/ Docker Compose)

## Requirements
- You need Docker to run it with the database
- Without the db, just node and npm is sufficient


To install, configure .env then do
```
npm install
```

To start, run
```
docker compose up --build --force-recreate
```
